    Create `src/components/admin/GrowthChart.jsx` — a `'use client'` component that renders a simple CSS bar chart for growth trends. No external charting library needed.

    **Props:**
    - `title` (string) — section label like "Weekly Trends" or "Monthly Trends"
    - `metrics` (array) — each item: `{ label, current, previous, unit? }` where label is metric name (e.g., "New Answers"), current/previous are numeric values, unit is optional suffix (e.g., "%")

    **For each metric, render:**
    - Label on the left
    - Two horizontal bars: "Previous" (warm-200 bg) and "Current" (green-500 if growth, red-400 if decline, warm-400 if flat)
    - Bar widths proportional to max value across current+previous for that metric
    - Percentage change badge: `+15%` in green or `-8%` in red, `0%` in warm
    - Actual numbers displayed: "42 → 58"

    **Styling:**
    - Container: `bg-white border border-warm-200 rounded-lg p-5`
    - Title: `text-sm font-semibold text-warm-500 uppercase tracking-wide mb-4`
    - Each metric row: flex column with label, bars, and change indicator
    - Bars: `h-3 rounded-full` with inline width style based on value proportion
    - Use Tailwind classes only (no CSS-in-JS or external libraries)

    **Edge cases:**
    - Previous is 0: show "+100%" or "New" badge in blue
    - Both are 0: show "—" in warm-400
    - Current is 0 but previous > 0: show "-100%" in red
  </action>
